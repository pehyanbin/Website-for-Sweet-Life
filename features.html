Implemented Features

1. Highlighted Menu Item

The navigation menu highlights the current page to provide visual feedback to users.

Implementation: This feature is implemented across all pages. You can see it in action by navigating between pages.

2. Adaptive Credit Card Length

The credit card number field adapts its validation based on the selected card type (16 digits for Visa/MasterCard, 15 digits for American Express).

Implementation: This feature is implemented on the Order page when "Pay Online" is selected.

3. Form Validation

All forms include comprehensive validation with inline error messages.

Implementation: This feature is implemented on both the Registration page and Order page.

4. Tooltip

Tooltip when hovering over the "Pay Online" CVV field checkbox to explain the payment method.

Implementation: This feature is implemented on the Order page.

5. Input Format Validation

Specific fields like email and contact number are validated to ensure the data format is correct before submission.

Implementation: This is applied on the Order page and the Registration page.

6. Local Storage

The registration and order forms automatically save your input to your browser's local storage. If you refresh the page or accidentally close the tab, your data will be re-populated, so you don't lose your progress.

Implementation: This feature is active on the Registration page and the Order page.
